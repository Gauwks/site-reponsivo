Programação Web I

A programação web é uma habilidade essencial para criar sites e aplicativos web interativos. Aqui estão os passos básicos para começar a aprender a programação web: HTML (HyperText Markup Language). O HTML é a espinha dorsal da web e é usado para criar a estrutura e o conteúdo básico de uma página web.

Nessa matéria somos lecionados pelo professor William Ribela, que nos ensina diversas coisas sobre HTML e CSS. Eles nos avalia de acrodo com os trabalhos e atividades passados e desenvolvidos em sala.

HTML

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo. Em informática, a Standard Generalized Markup Language é uma linguagem de marcação de padrão internacional, não proprietário e de código aberto

CSS

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML.
